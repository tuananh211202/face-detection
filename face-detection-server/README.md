# Python
## Install Python
### https://www.python.org/
## Install Pip
### `sudo apt install python3-pip`

# Create virtualenv
## Windows
### `.\myprojectenv\Scripts\activate`
## MacOS/Linux
### `source myprojectenv/bin/activate`

# Install dependency
### `pip3 install -r requirements.txt`

# Run server
### `python server.py`
Server runs at http://localhost:5000
